<p>Als er iets fout zit in code, geeft Python meestal een foutmelding. Dit zijn vaak "syntax fouten", die aangeven dat er iets fout zit in de vorm van je code (bijvoorbeeld, voor de code hierboven werden <span style="font-family: &quot;Comic Sans MS&quot;, sans-serif;">SyntaxErrors</span> gerapporteerd). Er zijn ook "runtime errors", die aangeven dat je code op zich syntactisch correct lijkt, maar dat er iets fout is gegaan bij de uitvoering ervan. Een goed voorbeeld is de <span style="font-family: 
        Comic Sans MS, sans-serif;">ZeroDivisionError</span>, die aangeeft dat je probeerde te delen door nul (wat niet mag, zoals je weet). Schrijf een kort programma dat zo'n fout genereert als je het uitvoert.</p>
