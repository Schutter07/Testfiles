<p>In de statistiek wordt de binomiaalco&euml;ffici&euml;nt &quot;n boven k,&quot; waarbij n groter dan of gelijk aan k moet zijn, berekend als n!/(k! * (n - k)!), waarbij n! de faculteit van n is. Zoals ik in hoofdstuk 7 heb uitgelegd: de faculteit van een positief geheel getal is dat getal, vermenigvuldigd met alle positieve gehele getallen die kleiner zijn (exclusief nul). Je schrijft de faculteit als het getal met een uitroepteken erachter. Bijvoorbeeld, 5 faculteit is 5! = 5 * 4 * 3 * 2 * 1 = 120. Als je alle opgaves tot nu toe gemaakt hebt, heb je hier code voor geschreven. Schrijf een functie die de binomiaalco&euml;ffici&euml;nt voor de twee parameters die de functie krijgt, en die de waarde retourneert. Schrijf de code op zo&rsquo;n manier dat hij als module in een ander programma kan worden opgenomen (dus test de functie via een main() functie zoals hierboven is uitgelegd).</p>
<p>Noem deze functie <span style="font-family: Comic Sans MS, sans-serif;">binom(n, k)</span>.</p>
